Como publicar no GitHub Pages (3 passos)
----------------------------------------
1. Crie um repositório novo no GitHub, por exemplo: lab-avaliacao-provisorio
2. Carregue (upload) todos os ficheiros deste ZIP para a raiz do repositório.
3. Vá a Settings → Pages → Branch: main (ou default) / folder: root → Save.
   A URL será algo como: https://SEU_UTILIZADOR.github.io/lab-avaliacao-provisorio/
